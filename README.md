# AMPGuard 🛡

AMPGuard is a lightweight Web Pentest Scanner built with Python and PySide6.

## ⚠️ Ethical Use Warning

AMPGuard is designed for **authorized security testing only**.  
You must have explicit written permission from the website owner before scanning.  
Unauthorized use may violate laws (e.g., Computer Fraud and Abuse Act).  
The author assumes no liability for misuse.

## Features

- Recursive crawler
- SQL Injection detection
- Reflected XSS detection
- Security header analysis
- Directory brute-force
- Interactive Dashboard
- Vulnerability severity chart
- Report generation
- Multi-target scanning

pip install -r requirements.txt
python main.py
